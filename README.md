# Testimonial-Slider
A simple testimonial slider can be built using HTML for structure, CSS for styling, and JavaScript for sliding behavior.

Testimonial Slider

📌 Overview

This project is a simple testimonial slider built using HTML, CSS, and JavaScript. It allows users to view multiple testimonials with smooth transitions and navigation controls.


🚀 Features

* Automatic sliding of testimonials
* Manual navigation using Previous and Next buttons
* Smooth fade transition effect
* Clean and responsive design


📁 Project Structure
index.html   → Main structure of the slider  
style.css    → Styling and layout  
script.js    → Slider functionality  
🛠️ How It Works

* All testimonials are placed inside a container (.slides).
* Only one testimonial is visible at a time using the active class.
* JavaScript updates the active slide when:
    * The user clicks navigation buttons
    * The auto-slide timer runs every 3 seconds


▶️ How to Run

1. Download or copy all three files:
    * index.html
    * style.css
    * script.js
2. Keep them in the same folder
3. Open index.html in any web browser


✏️ Customization

* Edit testimonials inside index.html
* Change slide timing in script.js:
setInterval(..., 3000); // Change 3000ms to your preferred time
